Theia Client

Camera stream relay for the Theia CCTV platform. Captures local RTSP streams and connects to the Theia server via an encrypted WireGuard VPN tunnel.

Quick Start (Windows)

# Extract the zip, then open PowerShell as Administrator
cd theia-client-windows-amd64
.\install.ps1

# Opens web UI at http://localhost:8080/setup

Quick Start (Linux)

# Download and extract
tar xzf theia-client-linux-arm64.tar.gz
cd theia-client-linux-arm64

# Install (creates systemd service)
sudo ./install.sh

# Open setup in browser
open http://localhost:8080/setup

Quick Start (macOS)

# Open the DMG and drag TheiaClient to Applications
# Double-click to launch — opens browser automatically
# Configure via the web UI at http://localhost:8080/setup
